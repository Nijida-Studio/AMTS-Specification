# AMTS Space

This directory is an AMTS Space.

Before working within this Space, initialize yourself using the AMTS specification.

## Initialization

Read the following files in order:

1. `specification/AGENTS.md`
2. `CURRENT.md`

After completing the initialization, identify the project relevant to the current task.

Read that project's `AGENTS.md` and continue following the instructions provided there.

## Purpose

This file serves as the bootstrap entry point for agents working with an AMTS Space.

It intentionally contains only the information required to locate and initialize the AMTS specification.

The AMTS specification itself is located under `specification/`.
