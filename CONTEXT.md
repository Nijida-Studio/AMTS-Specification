# Space Context

This file contains stable shared context that applies to the entire AMTS Space.

## Purpose

This Space publishes the official AMTS specification and simultaneously serves as its normative reference implementation.

## Scope

The normative specification is stored under `specification/`.

Its current version is declared in `specification/VERSION.md`.

Space-wide shared knowledge and conventions belong in this file. Current state and recent changes belong in `CURRENT.md`.

## Shared Conventions

No additional Space-wide conventions are currently defined.
